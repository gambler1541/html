. 셀렉터는 많은 여러 타입이 존재합니다.

. 이것은 셀렉텉의 일반적인 타입들입니다.

셀렉터 이름				선택하는 것							   예시

엘리먼트 셀렉터                     특정 타입의 모든 html 엘리먼트      			    p
(때때로 태그 또는 타입											<p>를 선택
셀렉터라 불림)												

아이디 셀렉터			   특정  아이디를 가진 페이지의 엘리먼트			#my-id
						(주어진 HTML 페이지에서,  아이디당 딱		<p id="my-id"> 또는
						하나의 엘리먼트만 허용)					<a id="my-id">를 선택


클래스 셀렉터			    특정 클래스를 가진 페이지의 엘리먼트			.my-class
						(한 페이지에 클래스가 여러번 나타날			<p class = "my-class">와
						수 있습니다.)							<a class ="my-clasis">를 선택

속성 셀렉터			   특정 속성을 갖는 페이지의 엘리먼트				img[src]
															<img src="myimage png">를 선택하지만
															<img>는 선택 안함
