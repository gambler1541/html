Hyper Link(하이퍼 링크)

웹 사이트는 여러개의 HTML 페이지를 가지고 페이지들이 상호 연결하는 방식으로
전체를 구성합니다.
HTML 페이지에서 다른 HTML 페이지를 연결하는 고리를
Hyper link 라고 부릅니다. 하이퍼 링크는 텍스트나 이미지에
다른 웹 페이지의 주소를 달아서 만드빈다.
하이퍼 링크는 같은 사이트의 다른 html 페이지 뿐만 아니라,
다른 사이트에 있는 웹 페이지를 연결하기도 합니다.

하이퍼 링크를 따라 다른 웹 페이지로 이동하느 ㄴ것을 항해 Navigation이라고
부릅니다. 
또한 하이퍼 링크는 링크라고 부르기도 합니다.


 하이퍼 링크 만들기

 <a href="URL" target ="self"></a>

 . href : 이동할 html 페이지의 URL, 또는 같은 페이지의 html, 다른 앵커 이름
